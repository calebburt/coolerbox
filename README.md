# DCC Utils

DCC Utils adds dark mode and other themes for sites commonly used by DCC. It was inspired by Coolbox, (T0RNATO/coolbox_extension_new).

## Coolerbox

Coolerbox is based on Coolbox. It lets you customise Schoolbox. It provides many themes, including dark mode. It also lets you rearrange the widgets on the home screen.

## Coolerspace

Coolerspace lets you customize Mathspace. It provides the same rich set of themes as Coolerbox, but with lots of transparency. In the future, it will support rearranging the widgets on the home screen.

## Coolerolo

Coolerolo does not exist yet, but will let you customize Edrolo.
